## Scenerio:
Ministry of Science Technology and Innovation in Abia state has an ICT training center with 100 Windows workstations (Windows 10/11 Pro) connected via a Local Area Network (LAN). 
They also have a file server (Windows Server 2022) storing ThriveAbia students and financial data. 
Recently, a staff member’s computer got infected with ransomware through a phishing email, spreading to three other PCs.

**Your task**: Redesign their LAN architecture with security in mind, explaining how to protect all Windows machines on that LAN.
